# Task Completion Checklist

After completing a coding task, ensure the following:

1.  **Linting**: Run `npm run lint` and fix any errors.
2.  **Build**: Run `npm run build` to ensure no TypeScript or build errors.
3.  **Formatting**: Format code if a formatter is available.
4.  **Unused Code**: Remove any unused imports, variables, or comments.
5.  **Verification**: Verify the changes locally (e.g., `npm run dev`).
6.  **Documentation**: Update documentation if necessary.
